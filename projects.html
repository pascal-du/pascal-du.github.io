1. Algorithm Visualizer (Spring 2024)

Designed and implemented a web‐based visualization tool for classic sorting and graph algorithms. Users can select an algorithm (e.g., quicksort, Dijkstra’s) and observe step‐by‐step animations illustrating internal operations. The goal was to aid novice learners in understanding algorithm mechanics.

Technologies: JavaScript, D3.js, HTML5 Canvas, CSS3.

2. Personal Blog Platform (Summer 2023)

Built a minimalist blogging platform from scratch, allowing authenticated users to publish, edit, and delete posts. Implemented a relational database schema with PostgreSQL and an Express.js backend. Emphasized security by sanitizing inputs and implementing CSRF protection.

Technologies: Node.js, Express.js, PostgreSQL, EJS, Bootstrap.
